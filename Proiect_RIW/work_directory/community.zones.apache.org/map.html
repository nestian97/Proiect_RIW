Apache Public Committers Map

This map shows locations of the ASF committers, who have chosen to publically share their location in their FOAF files.

Predefined views:





This map is updated every few hours based on committer FOAF files with locations in. If you're an ASF committer and you'd like to be shown here, please see this ASF Committers FOAF information page and generater for details on what your FOAF file needs to contain and where it needs to live.

If you are interested, you can also list yourself as a Local Mentor or as a Local Speaker, which will allow people to search for you here.


To see somewhat how this map has changed over time, please see the map of current public locations, filtered by the year someone became a committer.
